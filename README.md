# Video Game Rental System

## Setup
1. Install MongoDB
2. Clone repository
3. Install dependencies:
   ```bash
   cd backend
   npm install
   cd ../frontend
   npm install
   ```
4. Start servers:
   ```bash
   # Terminal 1
   mongod
   # Terminal 2
   cd backend
   node server.js
   # Terminal 3
   cd frontend
   npx http-server
   ```
